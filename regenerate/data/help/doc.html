Documentation

Documentation in regenerate is written in restruturedText format. Each document can have multiple pages, and each page can be tagged with tags.

reStructuredText

reStructuredText is a text based markdown language that allows complex formating to be done in basic text. See the reStructuredText Quick Reference for more details.

Pages

Pages appear as separate tabs. Clicking on the Add button in the upper right corner of the editor will add a new named page to the document. The selected page can be deleted by clicking on the Remove button.

Tags

Each page can be tagged with labels. The document editor does not set any policy on the tags. Multiple tags may be added.

Tags can be used to (among other things):

  • Document restrictions and confidentiality
  • Organization
  • Display control

Tags appear on the bottom of the page. They are added by clicking on the Add Tag button in the upper corner of the window. Tags are removed by clicking on the close button on the tag itself.
