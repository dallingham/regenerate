Getting Started

Regenerate has three levels of data, each of which has a tab on the left side of the screen. The lowest level is the Register Set, where individual registers are defined. The second level is the Block, where related registers set instances are grouped together into a logical block. The last level is the Top Level which groups blocks together into a top level design.

Currently, there are no blocks defined. Until there are, block instances cannot be added to the top level. The next step is to either:

  • Create new blocks under the Block tab by clicking on the new block button in the lower left corner of the Block tab.
  • Load an existing block from a .regb file by clicking on the add block button in the lower left corder of the Block tab. This will add the block and and register sets that it uses to the project.

Once blocks have been defined, you can add instances of the blocks to the top level by clicking on the Add button to the right. This will display a menu of the available blocks that can be added. Selecting one of these will add the instance to the top level, where it can be configured.
