No Register Sets Exist

Currently, there are no register sets defined. Until there are, register set instances cannot be added to this block. The next step is to either:

  • Create new register set under the Register Sets tab by clicking on the new register set button in the lower left corner of the Register Sets tab.
  • Load an existing register set from a .regp file by clicking on the add register set button in the lower left corner of the Register Sets tab.

Once register sets have been defined, you can add instances of the register sets to the block by clicking on the Add button to the right. This will display a menu of the available register sets that can be added. Selecting one of these will add the instance to the block, where it can be configured.
